Design Concept

I found the challenge to design possible solutions for these key pain points, while keeping in mind the following criteria:

  • Provides a game identical to August 17 celebration in general
  • Creating a sense of fun and competition that is the same as the independence race when playing the game
  • Make features simple and easy to use for users in concise time limit

Solution 1

Determining popular games that often followed by the community. These are the following games: panjat pinang, makan kerupuk, balap larung, paku bottle, and sepak bola sarung. From here we try to make every object in the game look exactly like that.

Solution 2

Provide rewards for the top 10 ranked users from the accumulated scores of all games, so that users are interested in playing and grow a sense of competitiveness between other users. Moreover, set the independence day song as BGM in game to make the atmosphere more real.

Solution 3

Action button in the game are simply adjusted. Like designing a game that can be played with just a single click, making it easier for all types of users when playing.
